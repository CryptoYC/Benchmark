# Benchmark是什么

为了让投资者透明化地了解一个区块链项目的技术情况，EVOLABTech从共识、安全、性能、技术创新、代码和经济模型进行研究，推出了区块链项目的技术尽调报告EVOLAB Benchmark。

若有想法或意见请向我们issue或者加入技术社区，非常感谢 :heart:
